Инструменты XQuery

Очень быстро появилось множество инструментов для работы с XQuery. Многие из них являются freeware или имеют открытый код. Ниже приводится далеко не полный список, взятый из домашней страницы XQuery на W3C:

  • Altova XML Query Processor - Altova Company
  • AquaLogic Data Services Platform - BEA Corp.
  • XStreamDB - Bluestream Database Software Corp.
  • XQEngine - Fatdog Software (GPL)
  • RainbowCore - Worcester Polytechnic Institute's
  • DataDirect Xquery - DataDirect
  • Qexo (Kawa-Query) - GNU
  • GoXML DB - XML Global
  • Nux - Berkeley Lab
  • NeoCore XMS - Xpriori
  • eXist - GNU
  • Qizx/open - AXYANA Software (GPL)

Последний из них мы (наугад и, как оказалось, очень удачно) выбрали для целей нашей презентации.
