Выбрать название товара, которого в 1990 г. было продано максимальное количество экземпляров

Не самый простой запрос

Запрос: 
 1  let $x:= <z> {
 2    for $y in 
 3      <y>{for $b in 
 4        doc('file:///c:/KING/kingx/king_corp.xml')//customers/
 5        distinct-values(//product_id)
 6        return <x> {
 7          <id> {$b} </id>,
 8          for $a in 
 9            doc('file:///c:/KING/kingx/king_corp.xml')//customers//
10            order[substring-before(order_date,'-')='1990']/item
11            where $a/product_id=$b
12            return $a/quantity
13          } </x>
13      } </y>
14      /x
15      order by $y/sum(quantity)
16      return ($y/id,$y/sum(quantity))
17    }</z>
18  let $z:=$x/id[last()]
19  for $a in doc('file:///c:/KING/kingx/king_corp.xml')//products/product

Результат:

Попробуем отследить выполнение этого запроса

  • Самый внутренний оператор for с переменной цикла $a (8-12) без оператора where (11) отбирает те заказы, которые были сделаны в 1990 и выбирает из них количества.
  • Оператор for с переменной цикла $b (2-5) выбирает все коды товаров.
  • В возвращаемом им значении оператор конструирует элемент x (6-13), в который вложен элемент id, тело которого - id товара, и количество проданного товара, выбранное вложенным оператором for (8-12). За счет where (11) количество "естественным образом" соединено с соответствующим id товара.
  • Все элементы x, сформированные на предыдущем шаге, вкладываются в элемент y (3,13).
  • Этот элемент y является источником для оператора for с переменной цикла $y (2-16). Этот оператор выбирает каждый вложенный элемент x (14) и вычисляет для него сумму значений quantity. Он возвращает id товара и суммарное количество для него (16), упорядоченные по возрастанию суммарного количества (15).
  • Все, что сформировал предыдущий оператор for, вкладывается в элемент z и запоминается в переменной $x (1,17).
  • Из переменной $x извлекается последний id (т.е., id товара с максимальным количеством) и запоминается в переменной $z (18).
  • Наконец, последний оператор for (19-21) выбирает название того товара, id которого совпадает с запомненным в переменной $z.
