Составить отчет по работе отделов продаж, в котором указать:

  • id отдела
  • город
  • список покупателей
  • общую сумму продаж

Это та же задача, что и в предыдущем примере, но на сей раз мы сформируем результат в виде HTML-страницыю
Запрос: 
 1 element html{
 2   element b{"Department report"},
 3   element table{
 4     attribute border{"1"}, 
 5     let $x:={
 6       for $a in doc('file:///c:/KING/kingx/king_corp.xml')//department
 7       where $a/name='SALES'
 8       return $a }
 9     for $a in $x//department
10     return ( 
11     element tr{
12       element td{attribute colspan {3},'Depatrment',$a/data(@id)}
13     },
14     element tr{
15       element td{ attribute valign{"top"},$a/location/text()},
16       element td{
17         for $b in doc('file:///c:/KING/kingx/king_corp.xml')//customers/customer
18         for $c in $a//employee
19         where $c/data(@id)=$b/data(salesperson)
20         return ($b/name/text(),element br {})
21       },
22       element td{
23         attribute valign{"top"}, 
24         let $y:={
25           for $b in doc('file:///c:/KING/kingx/king_corp.xml')//customers/customer
26           for $c in $a//employee
27           where $c/data(@id)=$b/data(salesperson)
28           return element t {sum($b/order/total)} }
29         for $d in $y return sum($d//t)
30       }
31     })
32   }
33 }
Результат:

Вы можете посмотреть в браузере результат, сохраненный в файле report.html.

Выполнение запроса:

  • Корневым элементом формируемого отчета будет элемент html (1-33).
  • Первым в корневой элемент вкладывается элемент b (полужирный шрифт) с заголовком (2).
  • Далее формируется элемент table (3-32), в котором:
    • содержится атрибут border со значением 1 (4);
    • формируется в переменной $x элемент x, в который вложены элементы department всех отделов, имеющих название SALES;
    • для каждого элемента department из переменной $x:
      • создается элемент tr, в который вкладывается элемент td с атрибутом colspan и номером отдела (11-13) - это строка заголовка отдела;
      • создается элемент tr (14-31), в который вкладываются:
        • элемент td с атрибутом valign и значением местоположения отдела (15);
        • элемент td, тело которого формируется (16-21) из названий покупателей, связанных с данным отделом через обслуживающих их продавцов, после каждого названия покупателя вставляется пустой элемент br;
        • элемент td с атрибутом valign, значение элемента вычисляется (22-30) как сумма продаж по отделу (как и в предыдущем примере.
