Вычисляемый конструктор

В вычисляемом конструкторе используются ключевые слова для создания элементов и атрибутов. Значения элементов и атрибутов задаются либо константами, либо выражениями, в том числе, это могут быть выражения FLOWR.

Пример: 

element employee {
  attribute id {"2265"},
  element name {"Edsger W. Dejkstra"},
  element salary {"4500.00"},
  element commission {"1000"},
  element hire_date {"1967-01-10"},
  element function {"RESEARCH WORKER"},
  element manager_id {"7250"}
}
