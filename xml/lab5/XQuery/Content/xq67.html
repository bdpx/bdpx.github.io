Выбрать коды тех отделов, в которых есть КАКИЕ-НИБУДЬ из должностей, имеющихся в отделе 23

Запрос: 
let $x:= <x>{
  for $a in  
    doc('file:///c:/KING/kingx/king_corp.xml')//department[@id='23']
  return $a
}</x>
for $c in    
    doc('file:///c:/KING/kingx/king_corp.xml')//department[@id!='23']
  where some $h in $x//function
  satisfies; $h=$c//function
return $c/data(@id)
Результат:

Еще квантор.

Оператором let создается список должностей отдела 23 и запоминается в переменной $x. Затем перебираются (for) отделы (кроме 23) и выбираются те, у которых какая-нибудь должность входит в список в переменной $x
