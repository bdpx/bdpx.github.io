Составить отчет по работе отделов продаж, в котором указать:

  • id отдела
  • город
  • список покупателей
  • общую сумму продаж
Запрос: 
 1 <report> {
 2   let $x:=<x>{
 3     for $a in doc('file:///c:/KING/kingx/king_corp.xml')
 4         //department
 5     where $a/name='SALES'
 6     return $a
 7     }</x>
 8   for $a in $x//department
 9   return <department>{$a/@id} {
10     $a/location,
11     <customers> {
12       for $b in doc('file:///c:/KING/kingx/king_corp.xml')
13          //customers/customer
14       for $c in $a//employee
15       where $c/data(@id)=$b/data(salesperson)
16       return $b/name    
17       } </customers>,
18     <total> {
19       let $y:=<y>{
20         for $b in doc('file:///c:/KING/kingx/king_corp.xml')
21           //customers/customer
22         for $c in $a//employee
23         where $c/data(@id)=$b/data(salesperson)
24         return <t>{sum($b/order/total)}</t>
25         }</y>
26       for $d in $y return sum($d//t)
27       }</total>
28     } </department>
29 }</report>
Результат:

Выполнение запроса:

  • Все, что мы сконструируем, будет вложено в корневой элемент report (1, 29).
  • Формируем и сохраняем в переменной $x элемент x, в который вложены элементы department всех отделов, имеющих название SALES.
  • Для каждого вложенного элемента department формируем выходной элемент department (9 - 28), в который вкладываются:
    • элемент location из исходного department (10);
    • элемент customers (11,17), в который вкладываются:
      • элементы name (12-16) - имена покупателей, для которых значение в элементе salesperson равно значению атрибута id какого-то из элементов employee, вложенных в данный department
    • элемент total (11,17), для вычисления которого:
      • формируется и сохраняется в переменной $y (19-25) элемент y, в который вложены элементы t, содержащие суммарный объем закупок по каждому покупателю;
      • вычисляется общая сумма по всем покупателям (26)
