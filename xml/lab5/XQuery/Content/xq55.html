Оператор return

создает результат выражения FLWOR для данного кортежа.

Пример:

Выбрать имена всех клерков

Запрос: 
for $a in doc('file:///c:/KING/kingx/king_corp.xml')//employee
  where $a/function='CLERK'
return <clerk> { data($a/name) } </clerk>
Результат:

Именно здесь, как правило, конструируется результат. В данном случае мы создаем новый элемент. Про конструирование результата мы еще поговорим ниже. Пока отметим, что фигурные скобки применяются для конструирования элемента, круглые (в предыдущих примерах) - для конструирования последовательности.
