Выбрать однофамильцев

Запрос: 
let $x:=<x> {
  for $a in 
    doc('file:///c:/KING/kingx/king_corp.xml')
    //employee
  for $b in
    doc('file:///c:/KING/kingx/king_corp.xml')
    //employee
 where ($b/tokenize(name,' ')[3] =
                         $a/tokenize(name,' ')[3])
         and ($b/name!=$a/name)
 order by $b/tokenize(name,' ')[3]
 return $b/name
} </x>
return distinct-values($x/name)
Результат:

Два оператора for создаются два списка имен-фамилий - $a и $b. Из второго списка ($b) выбираются те люди, для которых в первом списке ($a) есть пара с совпадающей фамилией и не совпадающим именем.

distinct-values() здесь употребляется потому, что если есть более двух однофамильцев, то каждый из них выберется несколько раз.
