XPath - язык для адресации частей XML-документа

Мы начнем с рассмотрения языка XPath, который является частью языка XQuery.

Вот основные свойства XPath:

  • XPath поддерживает числовые выражения, операции равенства и сравнения, и булевы выражения.
  • В результате обработки выражения получается объект, относящийся к одному из четырех основных типов:
    • набор узлов - неупорядоченный набор узлов без дубликатов
    • булево значение - true или false
    • число - число с плавающей точкой
    • строка - последовательность символов
  • Обработка выражений осуществляется, отталкиваясь от контекстного (текущего) узла
  • Главным выражением XPath является выражение адресации
