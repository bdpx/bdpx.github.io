Индивидуальная работа: «Win32 API»

1 Цель: Изучение основ программирования под Windows.

Введение

2 Индивидуальное задание – это обязательный вид работы, который необходимо выполнить студенту для получения высокой оценки по результатам семестра. Структура и цели индивидуального задания 3 Основной целью индивидуального задания является углубленное изучение теоретических и практических вопросов разработки и реализации компьютерных программ.

Задание

  1. Разработать динамическую библиотеку для работы с прямоугольными матрицами и векторами.
  2. Реализовать и экспортировать из библиотеки функции для создания матриц и векторов, их уничтожения, умножения матрицы на матрицу, матрицы на вектор, транспонирования матрицы, вывода в файл для матрицы и вектора.
  3. Реализовать модуль отложенного импорта для вашей библиотеки.
  4. В основной программе для целей тестирования использовать все функции вашей библиотеки.
  5. Для подключения библиотеки использовать отложенный импорт.
