Файлы для работы с библиотекой MPI

  • workfiles.zip - Необходимые для работы файлы одним архивом
  • bcwmpi.lib - статическая библиотека-переходник для компоновки с Borland C++ Builder
  • compile.bat - пример как компилировать в Borland C++ Builder
  • cvwmpi.dll - динамическая библиотека
  • mpi.h - заголовочный файл для программ написанных на C (но не C++)
  • run.bat - пример как запускать параллельную программу
  • test.c - пример минимальной программы
  • test.pg - пример конфигурационного файла (для запуска параллельного коллектива на локальной машине)

  • wmpi1.2.zip - архив для инсталляции
